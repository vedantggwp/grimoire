# Code of Conduct

## Our Pledge

Contributors and maintainers agree to make participation in this project respectful, constructive, and harassment-free.

## Expected Behavior

- Be clear, kind, and specific.
- Assume good intent while still naming bugs and risks directly.
- Keep criticism focused on code, docs, behavior, and maintainability.
- Respect privacy when discussing local workspaces or generated knowledge bases.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without consent.
- Repeated off-topic disruption of issues or pull requests.
- Sharing secrets or private user content in public project spaces.

## Enforcement

Maintainers may edit, hide, or remove comments and may block participants who violate this code of conduct. Serious security or privacy concerns should be reported privately using the security policy.
