# Support

Grimoire is maintained as an open-source project. Public support happens through GitHub issues and pull requests.

Use GitHub issues for:

- reproducible bugs;
- unclear documentation;
- feature requests with concrete use cases;
- compatibility problems with Claude Code plugin installs or MCP clients.

Before opening an issue, please check existing issues and include the command, Node.js version, operating system, and a minimal reproduction when possible.

Do not post secrets, private workspace files, or proprietary knowledge-base contents in public support requests.
