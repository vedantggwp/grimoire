# One Year of MCP: November 2025 Spec Release

> Raw source preserved verbatim. Do not edit this file.
> Compiled into wiki articles during the ingest stage.

---

MCP celebrated its first anniversary on November 25, 2025, with a major specification release introducing significant enhancements for enterprise adoption, agentic systems, and developer experience.

## Key Ecosystem Metrics

- MCP Registry grew to nearly 2,000 entries, representing 407% growth since September 2025
- 58 maintainers supporting 9 core/lead maintainers in the steering group
- 2,900+ contributors in the Discord community with 100+ new contributors joining weekly
- Thousands of active MCP servers across diverse industries

## Major Features in November 2025 Release

### Task-Based Workflows (SEP-1686)

"Tasks provide a new abstraction in MCP for tracking the work being performed by an MCP server."

This experimental capability enables:
- Active polling for work status
- Multi-step operation management
- States: working, input_required, completed, failed, cancelled
- Session-based security boundaries

### Simplified Authorization (SEP-991)

URL-based client registration using OAuth Client ID Metadata Documents, eliminating friction from Dynamic Client Registration. Clients now provide IDs pointing to self-managed JSON documents describing client properties.

### Security & Enterprise Features

- SEP-1024: Client security requirements for local installations
- SEP-835: Default scopes definition

### Extensions Framework

Extensions provide optional, additive, composable protocol enhancements:
- Operate outside core specification
- Allow experimentation before full integration
- Preserve backward compatibility
- Include authorization extensions (SEP-1046, SEP-990)

### URL Mode Elicitation (SEP-1036)

Enables secure out-of-band credential flows, allowing users to authenticate in browsers without exposing credentials to clients.

### Sampling with Tools (SEP-1577)

Allows servers to implement agentic loops using client tokens. Enables:
- Tool calling in sampling requests
- Server-side agent reasoning
- Parallel tool execution
- Explicit capability declarations

### Developer Experience Improvements

- SEP-986: Standardized tool naming format
- SEP-1319: Decoupled request payloads from RPC definitions
- SEP-1699: SSE polling improvements
- SEP-1309: Enhanced SDK version management

## Governance Evolution

In December 2025, Anthropic donated MCP to the Agentic AI Foundation (AAIF), a directed fund under the Linux Foundation, co-founded by Anthropic, Block and OpenAI.

## Industry Adoption

Major organizations integrating MCP include GitHub, Stripe, Notion, Hugging Face, Postman, OpenAI, AWS, Google Cloud, Microsoft, Block, and Okta.

## Backward Compatibility

The release maintains full backward compatibility; existing implementations function unchanged while new features remain optional.
