Model Context Protocol Overview

mcpprotocolarchitecturejson-rpc
MCP is an open JSON-RPC 2.0 protocol connecting LLM applications to external tools, data, and services through a host-client-server architecture.

Overview

The Model Context Protocol (MCP) is an open protocol that standardizes how LLM applications integrate with external data sources, tools, and services. Created by Anthropic, donated to the Agentic AI Foundation under the Linux Foundation in December 2025.

MCP draws inspiration from LSP. It defines a universal way to connect LLMs with context, using JSON-RPC 2.0 over stateful connections.

How It Works

Host → creates Client instances → each connects to one Server. Servers expose resources, tools, and prompts. Communication uses stdio or Streamable HTTP.

Key Capabilities

• Standardized context exchange — single protocol for all integrations
• Security-first — explicit user consent for tool invocations
• Progressive negotiation — unused features cost nothing
