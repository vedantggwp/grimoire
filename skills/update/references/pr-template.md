# PR Template

The PR body is the digest verbatim (see `digest-template.md`) plus the
reviewer checklist and footer below. Title format:

```
Grimoire update — {short topic} — {YYYY-MM-DD}
```

Appended after the digest:

```markdown
---

## Review checklist

- [ ] New sources are legitimate and in scope (`raw/` diffs show exactly what was archived)
- [ ] Article changes are accurate to their sources
- [ ] New cross-links make sense (each one names its relationship)
- [ ] Freshness actions are right (anything marked checked was actually verified)

Merging this PR accepts the update. To reject a source or connection,
delete the change and (for connections) the exclusion entry keeps it from
coming back. After merging, re-run /grimoire:compile and restart any MCP
client serving this wiki — the server loads artifacts once at startup.

---
Generated by /grimoire:update
```
